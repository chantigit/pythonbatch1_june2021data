if :  This keyword used to check condition in true case.
===	
syntax:
		if  condition :
				
				statements

Ex1:check given number is +ve or not

if-else:These keywords used to check condition in true & false case.
========
syntax:
		if  condition :
				statement1
				statement2
				statement3
		else:
				statement4
				statement5
				statement6
Ex2:check given number is +ve or -ve

elif: (nested if-else) This keyword used to check multiple conditions in true & false cases.
======================

syntax:
	
	if condition1:
		statement1
	elif condition2:
		statement2
	elif condition3:
		statement3
	else:
		statement4
